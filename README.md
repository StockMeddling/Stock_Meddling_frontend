# Project Idea

## Problem
- 현재 시장에 존재하는 자산 관리 앱들은 단순 정보 제공에 그치는 '플랫폼' 역할에 한정
- 실질적인 **포트폴리오 관리 솔루션** 부재
- 개인화된 리스크 관리 가이드 부족
- 펀드 의존도 감소와 개인 주식 투자 증가 트렌드에 대응 미흡

## Solution
- 과학적 모델 기반의 포트폴리오 최적화를 통해 개인화된 리스크 관리 전략 제시
- 복잡한 금융 이론의 사용자 친화적 해석 및 적용
- 실시간 시장 정보와 연계한 동적 포트폴리오 조정 제안
